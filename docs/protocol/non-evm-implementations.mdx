# Alt-VM 实现

虽然我们的公共文档目前专注于 EVM 兼容性，但我们提供对其他生态系统的不同级别的支持。这些生态系统可能尚未实现完整的功能对等或完整的工具集成（例如 TypeScript CLI 支持）。我们正在积极努力扩展我们在这些平台上的文档和工具。

:::info
如果您对在这些生态系统上构建或将 Hyperlane 引入其他新生态系统感到兴奋，请在 [我们的 Discord](https://discord.gg/hyperlane) 上与我们联系！
:::

## CosmWasm

请查看 Hyperlane 的 CosmWasm 实现 [这里](https://github.com/many-things/cw-hyperlane) 由 [Mitosis 团队](https://mitosis.org/) 实现和维护。请按照 [本指南](https://github.com/many-things/cw-hyperlane/blob/main/DEPLOYMENT.md) 了解如何在 Cosmos 应用链上进行部署。我们在 2024 年 7 月已在 Celestia、Stride 和 Neutron 上上线。

## Sealevel

Hyperlane v3 现已在 Solana 和 Eclipse 上上线，更多 Solana Rollup 即将推出。您可以在 [这里找到 Sealevel 合约的实现](https://github.com/hyperlane-xyz/hyperlane-monorepo/blob/main/rust/sealevel/programs)。

## Cairo

Hyperlane 协议已由 [Pragma 团队](https://www.pragma.build/) 在 Starkware 中实现。您可以在 [这里找到正在进行的实现](https://github.com/astraly-labs/hyperlane_starknet)。

## Move

Hyperlane 已由 [Movement Labs](https://movementlabs.xyz/) 在 MoveVM 中实现。您可以在 [这里找到正在进行的实现](https://github.com/movementlabsxyz/hyperlane-monorepo)。这些 Move 合约已在 Movement 的 M1 测试网上上线。

## Sway

请在 [这里找到我们正在进行的 Sway 实现](https://github.com/hyperlane-xyz/fuel-contracts)。
