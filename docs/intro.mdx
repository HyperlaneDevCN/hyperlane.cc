# Hyperlane 简介

## 什么是 Hyperlane？

Hyperlane 是一个无需许可的跨链通信互操作协议。它支持不同区块链之间的消息传递和资产转移，无需依赖中心化中介机构或任何许可。

## 为什么选择 Hyperlane？

Hyperlane 的无需许可设计意味着任何人都可以[部署 Hyperlane](https://hyperlanedevcn.github.io/website/docs/deploy-hyperlane)到任何区块链环境，无论是第一层区块链、扩容方案还是应用链，并立即开始构建跨链应用程序——无需批准，无需中介。这使开发者能够完全控制其去中心化应用在多链环境中的运行方式，消除瓶颈并实现可扩展性。

Hyperlane 设计时考虑了模块化，其[跨链安全模块(ISM)](https://hyperlanedevcn.github.io/website/docs/protocol/ISM/modular-security)允许开发者根据应用程序的需求配置、组合和自定义其安全模型。
