# Cosmos SDK

:::warning
**当前状态:** Hyperlane Cosmos SDK 模块目前正在开发中,即将**发布**!
:::

## 概述

Hyperlane Cosmos SDK 模块使基于 Cosmos SDK 的区块链能够集成 Hyperlane 进行跨链通信。它为 Cosmos 链提供了一种无需许可的方式来与外部生态系统进行通信,包括 EVM 和其他已[部署 Hyperlane](https://hyperlane.cc/docs/reference/addresses/mailbox-addresses) 的生态系统。

虽然 Hyperlane 已经通过 [CosmWasm](https://github.com/many-things/cw-hyperlane?tab=readme-ov-file) 支持 Cosmos 链,但该模块通过使 Cosmos SDK 链(无论是否支持 CosmWasm)都能利用 Hyperlane 的互操作性功能,从而扩大了其可访问性。

## 主要特性

- **端到端互操作性解决方案**: 为基于 Cosmos SDK 的链实现了 Hyperlane 跨链互操作性框架的消息传递、传输和安全模块。
- **无需许可且开源**: 任何人都可以部署和配置该模块以满足其需求,无需依赖中介。
- **由 KYVE 构建**: 由 [KYVE](https://www.kyve.network/) 团队开发,旨在将 Cosmos 互操作性扩展到 IBC 之外。
