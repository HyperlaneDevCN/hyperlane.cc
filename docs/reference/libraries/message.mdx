# `Message` 库

`Message` 库帮助您格式化和解析用于 Hyperlane `Mailbox` 合约的跨链消息。

- **构造消息**: `formatMessage` 让您可以将所有关键字段打包成一个可发送的消息。
- **提取字段**: 函数如 `version`、`nonce`、`origin`、`senderAddress`、`recipientAddress` 和 `body` 帮助您提取消息中的特定部分进行处理。
- **获取消息 ID**: `id` 使用 `keccak256` 为每条消息生成唯一标识符。
