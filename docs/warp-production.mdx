# 如何将您的 Hyperlane Warp 路由部署投入生产

为了将您的 Hyperlane warp 路由部署投入生产，您必须[移除受信任的中继器 ISM](/guides/remove-trusted-relayer.mdx)
从您的 warp 路由中。此外，您还需要[转移所有权](/guides/transfer-warp-route-ownership.mdx)给您的 warp 路由。完成这些操作后，将您的 warp 路由添加到[注册表](https://github.com/hyperlane-xyz/hyperlane-registry/tree/main/deployments/warp_routes)中，以便进行审核并纳入网络及其集成。
